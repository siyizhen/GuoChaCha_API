# 果查查API

## 一、介绍
果查查API 是一个专注于 **苹果 & 安卓设备序列号/IMEI 查询**、**证件照回执办理** 的API服务平台。

- **官网地址**：`https://api.51gcc.com`

## 二、统一请求说明

- **请求地址**：`https://open-api.51gcc.com`
- **请求方法**：POST（`application/x-www-form-urlencoded`）
- **公共参数**：
  - `api_id`：接口 ID
  - `api_key`：用户中心 API 密钥
  - `api_test`：是否模拟请求返回示例数据（`1=是；0=否`）
  - `timestamp`：毫秒时间戳（用于验签）
  - `sign`：签名（用于验签）


## 三、调用示例

**PHP**
```php
<?php
    $url = 'https://open-api.51gcc.com';
    $params = ['sn' => 'YOUR_VALUE', 'api_id' => '1', 'api_test' => 'YOUR_VALUE', 'api_key' => 'YOUR_VALUE', ];
    $ch = curl_init();
    curl_setopt($ch, CURLOPT_URL, $url);
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($params));
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    $response = curl_exec($ch);
    curl_close($ch);
    echo $response;
```

**Python**

**Java**
```java
import java.io.*;
import java.net.*;
import java.util.*;

public class ApiClient {
    public static void main(String[] args) throws Exception {
        String url = "https://open-api.51gcc.com";
        Map<String, String> params = new HashMap<>();
        params.put("sn", "YOUR_VALUE");
        params.put("api_id", "1");
        params.put("api_test", "YOUR_VALUE");
        params.put("api_key", "YOUR_VALUE");
        
        URL obj = new URL(url);
        HttpURLConnection con = (HttpURLConnection) obj.openConnection();
        con.setRequestMethod("POST");
        con.setDoOutput(true);
        
        StringBuilder postData = new StringBuilder();
        for (Map.Entry<String, String> param : params.entrySet()) {
            if (postData.length() != 0) postData.append('&');
            postData.append(URLEncoder.encode(param.getKey(), "UTF-8"));
            postData.append('=');
            postData.append(URLEncoder.encode(param.getValue(), "UTF-8"));
        }
        
        try (DataOutputStream wr = new DataOutputStream(con.getOutputStream())) {
            wr.writeBytes(postData.toString());
        }
        
        int responseCode = con.getResponseCode();
        BufferedReader in = new BufferedReader(new InputStreamReader(con.getInputStream()));
        String inputLine;
        StringBuffer response = new StringBuffer();
        while ((inputLine = in.readLine()) != null) {
            response.append(inputLine);
        }
        in.close();
        System.out.println(response.toString());
    }
}
```
